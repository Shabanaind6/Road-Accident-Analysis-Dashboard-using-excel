# Road Accident Analysis Dashboard (Excel Project)

This project focuses on analyzing road accident data using Microsoft Excel to identify trends, high-risk locations, and key contributing factors. The dashboard provides clear visual insights to support data-driven safety analysis.

## Objectives
- Analyze road accident patterns and trends
- Identify high-risk areas and accident causes
- Present insights using an interactive Excel dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Conditional Formatting
- Data Cleaning & Analysis

## Project Files
- Road_Accident_Data.xlsx – Dataset
- Road_Accident_Dashboard.xlsx – Excel dashboard
- Road_Accident_Analysis_Presentation.pptx – Project presentation

## Key Insights
- Accident trends by location and time
- High-risk zones with frequent accidents
- Major contributing factors affecting accident rates

## Conclusion
This Excel-based dashboard helps transform raw accident data into meaningful insights, making it easier to understand patterns and support road safety decision-making.
